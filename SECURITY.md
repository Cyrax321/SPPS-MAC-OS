# Security Policy

## Scope

This repository is a **read-only peer-review artifact**.  
It is not a production library and has no active maintenance track.

## Reporting Issues

If you discover a security concern in the experimental code:

1. **Do not open a public GitHub issue.**
2. Email the contact address listed in the submitted manuscript.
3. Include a clear description of the potential issue and reproduction steps.

We will respond within 14 days for any review-period concerns.

## Supported Versions

| Version | Supported |
|---|---|
| Review snapshot (0.4.0) | ✅ Current |
| All earlier versions | ❌ Not supported |

## Reproducibility Contact

For reproducibility issues that are not security-related,
please open a GitHub issue using the **Reproducibility Issue** template.

For hardware-specific questions (non-Apple-Silicon results),
refer to `docs/cache_analysis.md` for explanation of M1-specific behaviour.
